# 🐘 Aprendiendo en PostgreSQL: Mejorar mis bases

¡Hola! 👋 Este repositorio es mi bitácora de aprendizaje mientras domino las bases de datos relacionales. Como aspirante a **Backend Developer**, entiendo que los datos son el corazón de cualquier aplicación, y este curso es parte de mi formación para construir sistemas confiables y escalables.

## 🎯 Objetivos del Repositorio
* **Consolidar la lógica SQL:** Más allá de escribir queries, busco entender la teoría relacional y la optimización.
* **Documentación Personal:** Tener un histórico de consultas complejas, funciones y triggers para consulta rápida.

## 🛠️ Stack de Desarrollo
Para este curso, he configurado un entorno de desarrollo sobre **Fedora Linux**:
* **Motor:** PostgreSQL (corriendo en contenedores de **Docker** para aislamiento total).
* **IDE:** DataGrip (licencia estudiantil) y pgAdmin4.
* **Control de Versiones:** Git & GitHub.

## 📂 Estructura del Proyecto
* `/01-fundamentos`: SELECT, filtros, operadores y funciones de strings.
* `/02-agrupaciones`: Group By, Having y funciones de agregación (proximamente).
* `/03-relaciones`: Joins, llaves primarias y foráneas (proximamente).
